13 Class
14  Foam::sodiumCrypto
15 
16 Description
17  Optional asymmetric encryption for boundary condition data using libsodium.
18 
19  Uses sealed boxes (crypto_box_seal) for anonymous encryption:
20  - Encryption requires only the recipient's PUBLIC key
21  - Decryption requires both PUBLIC and PRIVATE keys
22  - Provides confidentiality and integrity
23 
24  Key format:
25  - Keys are 32 bytes, represented as base64 strings for controlDict
26  - Public key can be freely shared in controlDict
27  - Private key is NEVER stored - entered via stdin at init time
28 
29  File format (.enc):
30  - 4 bytes: magic "ENC1"
31  - 4 bytes: flags (reserved)
32  - 8 bytes: original data size (little-endian)
33  - N bytes: sealed box ciphertext
34 
35  Workflow:
36  1. Generate keypair: spaceTimeWindowKeygen
37  2. Extraction: PUBLIC key in controlDict, BC data encrypted to .enc files
38  3. Init: spaceTimeWindowInitCase prompts for PRIVATE key (stdin, no echo)
39  and decrypts .enc files to plain format
40  4. Reconstruction: runs with decrypted data
41 
42  Usage in controlDict (extraction):
43  \verbatim
44  spaceTimeWindowExtract1
45  {
46  ...
47  encrypt true;
48  publicKey "base64-encoded-32-byte-public-key";
49  }
50  \endverbatim
51 
52 Note
53  This class compiles to no-ops if FOAM_USE_SODIUM is not defined.
54  Enable with: export FOAM_USE_SODIUM=1 before wmake
