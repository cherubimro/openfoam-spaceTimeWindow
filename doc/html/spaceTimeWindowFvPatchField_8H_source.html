32 Description
33  Boundary condition for space-time window reconstruction.
34 
35  Reads pre-computed face values from boundaryData directory structure.
36  Unlike timeVaryingMappedFixedValue, this BC:
37  - Does NO spatial interpolation (values are pre-computed for exact faces)
38  - Only does temporal interpolation between available timesteps
39  - Expects face values computed via linear cell-to-face interpolation
40 
41  The boundaryData structure:
42  \verbatim
43  constant/boundaryData/<patchName>/
44  points (optional, not used - no spatial mapping)
45  <time1>/
46  <fieldName> (face values as Field<Type>)
47  <time2>/
48  <fieldName>
49  ...
50  \endverbatim
51 
52 Usage
53  \table
54  Property | Description | Required | Default
55  dataDir | boundaryData directory path | no | constant/boundaryData
56  setAverage | adjust mapped field average | no | false
57  offset | offset value added to field | no | Zero
58  fixesValue | report to adjustPhi that values are fixed | no | true
59  \endtable
60 
61  Example of the boundary condition specification:
62  \verbatim
63  <patchName>
64  {
65  type spaceTimeWindow;
66  dataDir "constant/boundaryData";
67  setAverage false;
68  offset (0 0 0); // For vectors
69  fixesValue true; // Tell adjustPhi to exclude this patch
70  value uniform (0 0 0);
71  }
72  \endverbatim
